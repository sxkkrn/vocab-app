# 単語テストアプリ

- `index.html` : テスト画面
- `list.html` : 単語一覧画面
- `words_en.js` : 英語単語データ
- `words_de.js` : ドイツ語単語データ

## GitHub Pagesで公開
1. GitHubで新しいリポジトリを作る
2. この4ファイルをアップロード
3. Settings → Pages を開く
4. Build and deployment の Source を `Deploy from a branch` にする
5. Branch を `main`、Folder を `/ (root)` にする
6. 少し待つと公開URLが表示される
